# Tome-Ocean
A gamified, front-end web application that uses machine learning to provide highly personalized book recommendations

Key Features & Technologies:
•	Recommendation Engine: Developed a hybrid recommendation system using TensorFlow.js in the browser, calculating cosine similarities between user reading vectors and a 10,000+ book dataset to deliver real-time, accurate suggestions.
•	Immersive User Experience: Built a highly interactive ocean-themed UI utilizing HTML5 Canvas, Vanilla JavaScript, and CSS3, featuring custom particle rendering, fluid wave animations, and gamified character avatars.
•	Data & State Management: Engineered a responsive, seamless Single Page Application (SPA) architecture capable of searching, filtering by genre/mood, and storing customized reading shelves entirely on the client side without performance degradation.
